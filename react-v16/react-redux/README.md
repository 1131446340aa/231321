# 总结 redux的设计使用原则
1. store 是唯一的
2. 只有store能改变自己的内容
2. reducer 是一个纯函数

# redux 中核心的API
1. createState 创建store
2. store.dispatch 派发action
3. store.getState 获取store里面所有的数据
4. store.subscribe  订阅store数据的改变
